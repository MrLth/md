# 1. 盒子模型

## 1.1 组成

1. 内容 **content**
2. 填充 **padding**
3. 边框 **border**
4. 边界 **margin**

## 1.2 分类

	1. **IE盒子模型** **border-box**，width，height 包含 content、border、padding
 	2. **W3C**标准盒子模型 **content-box** ，width，height 只包含 content

## 1.3 box-sizing

​	在 IE8+ 的浏览器可以通过 box-sizing 来控制使用的盒子的模型，默认值为 **content-box**
